# DESIGN.md

## 气质与意象
午夜倒计时秒杀现场：红色数字倒数、金色光点闪烁、人群凝神等待的紧张感。一张干净的票据，右上角烫金印章，下方红色"抢"字戳印——权威、紧迫、价值感三重叠加。

## 视觉策略
- 图形语言：圆角卡片 + 微投影，模拟实体优惠券质感
- 倒计时区块用等宽字体、大号红色数字，营造紧迫感
- 券面采用"票据撕边"样式（虚线分割主券与副券区）

## 配色方案
- 主色 `#DC2626`（red-600）：抢券按钮、倒计时、关键标签 — 来源：传统年节红，激发行动欲
- 辅色 `#D97706`（amber-600）：价格数字、回兑标签、金色徽章 — 来源：金箔印章
- 背景 `#F9FAFB`（gray-50）：页面底色 — 来源：白色票据纸面
- 卡片 `#FFFFFF`：券卡片底色
- 文字 `#111827`（gray-900）：主文本
- 次要文字 `#6B7280`（gray-500）：说明文字

## 字体排版
- 标题：Inter / system-ui，600-700 weight
- 数字/倒计时：`font-variant-numeric: tabular-nums`，等宽数字
- 价格：24px+ 加粗，amber-600 色

## 动效与交互
- 抢券按钮：hover 放大 1.03 + 微弹动（cubic-bezier(0.34, 1.56, 0.64, 1)）
- 倒计时：每秒数字切换带 translateY 微滑入
- 卡片入场：fadeIn + translateY(8px)，stagger 80ms
- 页面切换：fade 过渡 200ms

## 页面结构
- 首页：顶部场次 Tab → 券卡片网格 → 倒计时居中
- 个人中心：头像区 + 认证状态条 → 余额卡片 → 券列表 Tab（待使用/待回兑/已完成）
- 管理端：左侧导航 → 右侧数据表格 + 操作按钮

## 设计禁忌
- 禁止蓝紫色系（科技感过强，与抢购紧迫感冲突）
- 禁止扁平无边框按钮（抢券场景需要明确行动区）
- 禁止小字号倒计时（需远距离可读）
- 禁止灰色抢券按钮（用 disabled 红底 + 不可点击文字替代）
