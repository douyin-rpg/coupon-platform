# DESIGN.md

## 气质与意象
抖音电商秒杀现场：霓虹红渐变横幅、金色倒计时跳动、卡片式商品瀑布流。手机屏幕上的限时抢购直播间氛围——热情、紧迫、高能量，同时保持信息清晰可读。像一张有质感的电子票据，左侧红色抢购标记，右侧金色价格，底部虚线撕切。

## 视觉策略
- 图形语言：圆角卡片（12px radius）+ 微投影（shadow-sm），模拟抖音商品卡质感
- 倒计时区块用等宽字体、大号红+金渐变数字，营造紧迫感
- 券面采用"票据撕边"样式（虚线分割主券与副券区）
- 首页顶部渐变横幅 Banner，参考抖音商城热卖区
- 卡片图片区占上半部分，下方信息区，类似抖音商品卡片

## 配色方案
- 主色 `#FE2C55`（抖音红）：抢券按钮、倒计时、关键标签、Tab 选中态 — 来源：抖音品牌红
- 辅色 `#FFC107`（金色/amber-500）：价格数字、回兑标签、金色徽章 — 来源：直播间金币
- 渐变 `linear-gradient(135deg, #FE2C55, #FF6B35)`：主按钮、Banner 背景
- 背景 `#F5F5F5`（gray-100）：页面底色 — 来源：抖音商城灰底
- 卡片 `#FFFFFF`：券卡片底色
- 文字 `#1A1A1A`（近黑）：主文本
- 次要文字 `#999999`（gray-400）：说明文字、时间
- 标签底色 `#FFF0F0`（淡红）：折扣/限时标签背景

## 字体排版
- 标题：Inter / system-ui，600-700 weight
- 数字/倒计时：`font-variant-numeric: tabular-nums`，等宽数字
- 价格：20px+ 加粗，#FFC107 色 + 深色底色衬托
- 场次 Tab：14px 加粗，选中态下划线红色

## 动效与交互
- 抢券按钮：hover 放大 1.03 + 微弹动（cubic-bezier(0.34, 1.56, 0.64, 1)）
- 倒计时：每秒数字切换带 translateY 微滑入
- 卡片入场：fadeIn + translateY(8px)，stagger 80ms
- 页面切换：fade 过渡 200ms
- 按钮 active：scale(0.97) 按压感
- 导航栏底部安全区：底部 Tab 固定，磨砂玻璃效果

## 页面结构
- 首页：顶部渐变 Banner（含品牌 logo + 倒计时）→ 场次 Tab → 券卡片网格（2列瀑布流）→ 底部导航栏（首页/我的）
- 个人中心：顶部渐变卡片（头像+昵称+认证状态）→ 钱包卡片（余额+提现）→ 功能入口网格 → 券列表 Tab
- 管理端：左侧深色导航 → 右侧白色卡片区域 + 数据表格 + 操作按钮

## 设计禁忌
- 禁止蓝紫色系（科技感过强，与抢购紧迫感冲突）
- 禁止扁平无边框按钮（抢购场景需要明确的行动区）
- 禁止小字号倒计时（需远距离可读）
- 禁止灰色抢券按钮（用 disabled 红底低透明度 + 不可点击文字替代）
- 禁止纯白无层次背景（至少用 gray-100 底色区分卡片与页面）
