# DESIGN.md

## 气质与意象
抖音电商官方风格：清澈天蓝渐变背景、浮动半透明光球动画、品牌Logo青紫渐变图标。整体氛围是信任感+科技感+活力，类似抖音电商官网（douyinec.com）的开放入驻页面——干净、专业、有呼吸感，同时保留抢购场景的紧迫节奏。

## 视觉策略
- Logo：使用抖音电商官方Logo（青紫渐变抖动音符图标 + "抖音电商"文字）
- 首页Hero区域：深蓝多段渐变背景 + 网格点阵纹理 + Aurora极光动画（模糊光晕+水平光带，3组不同速度循环）+ 倒计时磨砂玻璃卡片
- 圆角卡片（12px radius）+ 微投影（shadow-sm），保留商品卡质感
- 倒计时区块用等宽字体、大号数字，磨砂玻璃容器（backdrop-blur + inset shadow）
- 卡片图片区占上半部分，下方信息区，类似抖音商品卡片

## 配色方案
- 主色 `#1890FF`（抖音蓝）：导航栏、按钮主色、Tab选中态 — 来源：抖音电商官网主色
- 品牌渐变 `linear-gradient(135deg, #00D4FF, #7B61FF)`：Logo图标渐变、Hero区域装饰 — 来源：抖音品牌青紫渐变
- 辅色 `#FE2C55`（抖音红）：抢券按钮、倒计时、关键标签 — 来源：抖音品牌红
- 价格色 `#FF6B35`（活力橙）：价格数字、折扣标签 — 来源：直播间活力橙
- Hero背景 `linear-gradient(170deg, #060E1A 0%, #0A1A30 25%, #0D2244 50%, #091B35 75%, #060E1A 100%)`：深蓝极光渐变 — 来源：抖音电商官网Hero区优化版
- 页面背景 `#F5F7FA`（浅蓝灰）：页面底色
- 卡片 `#FFFFFF`：券卡片底色
- 文字 `#1A1A1A`（近黑）：主文本
- 次要文字 `#8C8C8C`：说明文字、时间
- 标签底色 `#E6F7FF`（淡蓝）：折扣/限时标签背景

## 字体排版
- 标题：Inter / system-ui，600-700 weight
- 数字/倒计时：`font-variant-numeric: tabular-nums`，等宽数字
- 价格：20px+ 加粗，#FF6B35 色
- 场次 Tab：14px 加粗，选中态下划线蓝色

## 动效与交互
- Hero区域：Aurora极光动画（3组模糊光晕+1组水平光带，不同速度/方向循环）+ 网格点阵纹理
- 抢券按钮：hover 放大 1.03 + 微弹动（cubic-bezier(0.34, 1.56, 0.64, 1)）
- 倒计时：磨砂玻璃容器 + inset高光 + 每秒数字切换带微滑入
- 卡片入场：fadeIn + translateY(8px)，stagger 80ms
- 页面切换：fade 过渡 200ms
- 按钮 active：scale(0.97) 按压感
- 导航栏底部安全区：底部 Tab 固定，磨砂玻璃效果

## 页面结构
- 首页：深蓝Hero区（品牌Logo+Aurora极光+网格点阵+倒计时磨砂卡片）→ 场次Tab → 券卡片网格 → 底部导航栏
- 公告列表：深蓝极光Header → 分类药丸Tab（各分类独立渐变色）→ 文章卡片（封面图/彩色分类图标+标题+摘要+浏览量）
- 公告详情：深蓝极光Header → 白色卡片（封面图+分类标签+标题+浏览量+内容）
- 个人中心：顶部蓝色渐变卡片（头像+昵称+认证状态）→ 钱包卡片（余额+提现）→ 功能入口网格
- 管理端：左侧深蓝导航 → 右侧白色卡片区域 + 数据表格 + 操作按钮

## 设计禁忌
- 禁止蓝紫色系过度堆砌（保持蓝系但需有明度/饱和度层次）
- 禁止扁平无边框按钮（抢购场景需要明确的行动区）
- 禁止小字号倒计时（需远距离可读）
- 禁止灰色抢券按钮（用 disabled 蓝底低透明度替代）
- 禁止纯白无层次背景（至少用 #F5F7FA 底色区分卡片与页面）
